## The Setup

A $14 billion company. 1,700 active projects in Autodesk Construction Cloud. When a new team member joins, someone has to add them to every relevant project. Manually. One click at a time.

When someone leaves, someone has to remove them. From every project. Manually.

When compliance asks "who has access to what?" -- nobody knows.

This is the enterprise ACC problem.

## The RAPS Solution

```bash
# Add user to ALL projects -- 1,700 projects, ~45 seconds
raps admin user add "$ACCOUNT" "newhire@company.com" --role project_admin

# Add to only 2024 projects
raps admin user add "$ACCOUNT" "newhire@company.com" \
  --role project_admin --filter "^2024-"

# Remove departing employee from everywhere
raps admin user remove "$ACCOUNT" "departing@company.com"

# Preview before executing
raps admin user add "$ACCOUNT" "newhire@company.com" --dry-run

# Clone permissions from template
raps admin folder set-permissions "$ACCOUNT" \
  --project "$NEW" --clone-from "$TEMPLATE"

# Export for audit
raps admin user list "$ACCOUNT" --format csv > audit.csv
```

Each of these commands replaces hours of manual work with a single line.
