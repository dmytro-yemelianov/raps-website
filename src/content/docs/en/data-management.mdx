# Data Management Commands

Browse hubs, projects, folders, and items in BIM 360 and Autodesk Construction Cloud (ACC).

![RAPS data management demo](/terminal/data-management-demo.gif)

> **Note:** Data Management commands require 3-legged OAuth. Run `raps auth login` first.

## Commands Overview

| Command | Description |
|---------|-------------|
| `raps hub list` | List accessible hubs |
| `raps hub info` | Get hub details |
| `raps project list` | List projects in a hub |
| `raps project info` | Get project details |
| `raps folder list` | List folder contents |
| `raps folder create` | Create a new folder |
| `raps folder rename` | Rename a folder |
| `raps folder delete` | Delete a folder |
| `raps folder permissions` | Show folder permissions |
| `raps item info` | Get item details |
| `raps item versions` | List item versions |
| `raps item create-from-oss` | Create item from OSS object |
| `raps item delete` | Delete an item |
| `raps item rename` | Rename an item |

## raps hub list

List all accessible BIM 360/ACC hubs.

```bash
$ raps hub list

└──────────────────────────────────────────┴─────────────────────────────┴───────┘
```

## raps folder list

List contents of a folder.

```bash
$ raps folder list b.proj-123 urn:adsk.wipprod:fs.folder:abc123
Folder Contents:
┌─────────┬────────────────────────────────┬───────────────────────────────┐
│ Type    │ Name                           │ ID                            │
├─────────┼────────────────────────────────┼───────────────────────────────┤
│ folder  │ Drawings                       │ urn:adsk...folder:def456      │
│ folder  │ Models                         │ urn:adsk...folder:ghi789      │
│ item    │ Project Schedule.xlsx          │ urn:adsk...item:jkl012        │
└─────────┴────────────────────────────────┴───────────────────────────────┘
```

**List top-level folders:**

```bash
raps folder list b.proj-123 --top
```

## raps folder create

Create a new folder.

```bash
$ raps folder create b.proj-123 urn:adsk...folder:parent --name "New Folder"
✓ Folder created successfully!
  Name: New Folder
  ID: urn:adsk.wipprod:fs.folder:xyz789
```

## raps folder rename

Rename a folder.

```bash
raps folder rename <PROJECT_ID> <FOLDER_ID> --name <NEW_NAME>
```

**Example:**

```bash
$ raps folder rename b.proj-123 urn:adsk...folder:abc123 --name "Revised Drawings"
✓ Folder renamed successfully!
```

## raps folder delete

Delete a folder.

```bash
raps folder delete <PROJECT_ID> <FOLDER_ID>
```

> **Warning:** Folder must be empty. Move or delete contents first.

**Example:**

```bash
$ raps folder delete b.proj-123 urn:adsk...folder:abc123
✓ Folder deleted successfully!
```

## raps item delete

Delete an item from a project.

```bash
raps item delete <PROJECT_ID> <ITEM_ID>
```

**Example:**

```bash
$ raps item delete b.proj-123 urn:adsk.wipprod:dm.lineage:abc123
✓ Item deleted successfully
```

## raps item rename

Rename an item (update display name).

```bash
raps item rename <PROJECT_ID> <ITEM_ID> --name <NEW_NAME>
```

**Options:**
- `--name, -n` - New display name

**Example:**

```bash
$ raps item rename b.proj-123 urn:adsk.wipprod:dm.lineage:abc123 --name "Updated Model v2.rvt"
✓ Item renamed to: Updated Model v2.rvt
```

## Common Workflows

### Browse project structure

```bash
# 1. List hubs
raps hub list

# 2. List projects in a hub
raps project list b.12345678-1234-1234-1234-123456789abc

# 3. List top-level folders
raps folder list b.proj-123 --top

# 4. Browse specific folder
raps folder list b.proj-123 urn:adsk.wipprod:fs.folder:abc123
```

### Export project data

```bash
# Export all hubs as JSON
raps hub list --output json > hubs.json

# Export projects
raps project list $HUB_ID --output json > projects.json
```
