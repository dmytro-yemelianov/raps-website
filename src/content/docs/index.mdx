## Quick Example

```bash
# Install via Cargo (Rust)
cargo install raps

# Configure credentials
raps config set client_id "your_client_id"
raps config set client_secret "your_client_secret"

# Test connection
raps auth test

# Upload and translate a model
raps dm upload ./model.dwg --translate --formats svf2
```
