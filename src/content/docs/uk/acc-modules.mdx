# ACC Module Commands

Manage extended Autodesk Construction Cloud modules: Assets, Submittals, and Checklists.

> **Note:** ACC module commands require 3-legged OAuth with project access. Run `raps auth login` first.

## Commands Overview

| Command | Description |
|---------|-------------|
| `raps acc asset` | Manage project assets |
| `raps acc submittal` | Manage project submittals |
| `raps acc checklist` | Manage project checklists |

---

## Asset Commands

Manage physical and digital assets tracked within a project.

| Command | Description |
|---------|-------------|
| `raps acc asset list` | List assets in a project |
| `raps acc asset get` | Get a specific asset |
| `raps acc asset create` | Create a new asset |
| `raps acc asset update` | Update an existing asset |
| `raps acc asset delete` | Delete an asset |

### raps acc asset list

```bash
raps acc asset list <PROJECT_ID> [--output FORMAT]
```

### raps acc asset create

```bash
raps acc asset create <PROJECT_ID> [OPTIONS]
```

**Example:**

```bash
$ raps acc asset list my-project-id --output json
$ raps acc asset get my-project-id asset-123
```
