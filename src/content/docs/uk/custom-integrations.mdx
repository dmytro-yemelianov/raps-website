# Налаштування Custom Integrations в ACC

Щоб RAPS міг працювати з адмініструванням облікового запису ACC — додавання та видалення учасників проєктів, оновлення ролей, керування правами доступу до папок і виконання масових операцій — ваш APS-застосунок має бути зареєстрований як **Custom Integration** у вашому обліковому записі ACC.

Це одноразове налаштування, яке виконує адміністратор облікового запису через веб-інтерфейс ACC.

> Англомовна версія цієї сторінки містить повний опис. Переклад буде додано найближчим часом.

## Пов'язані сторінки

- [`raps init`](/docs/init) — Майстер першого налаштування
- [`raps admin`](/docs/admin) — Довідник команд адміністрування
- [`raps auth login`](/docs/auth) — Повторна автентифікація
