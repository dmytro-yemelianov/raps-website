# Команди управління даними

Перегляд хабів, проєктів, папок та елементів у BIM 360 та Autodesk Construction Cloud (ACC).

![RAPS data management demo](/terminal/data-management-demo.gif)

> **Примітка:** Команди управління даними потребують 3-legged OAuth. Спочатку виконайте `raps auth login`.

## Огляд команд

| Команда | Опис |
|---------|------|
| `raps hub list` | Показати доступні хаби |
| `raps hub info` | Отримати деталі хаба |
| `raps project list` | Показати проєкти в хабі |
| `raps project info` | Отримати деталі проєкту |
| `raps folder list` | Показати вміст папки |
| `raps folder create` | Створити нову папку |
| `raps folder rename` | Перейменувати папку |
| `raps folder delete` | Видалити папку |
| `raps folder permissions` | Показати права доступу до папки |
| `raps item info` | Отримати деталі елемента |
| `raps item versions` | Показати версії елемента |
| `raps item create-from-oss` | Створити елемент з об'єкта OSS |
| `raps item delete` | Видалити елемент |
| `raps item rename` | Перейменувати елемент |

## raps hub list

Показати список усіх доступних хабів BIM 360/ACC.

```bash
$ raps hub list
Hubs:
┌──────────────────────────────────────────┬─────────────────────────────┬───────┐
│ ID                                       │ Name                        │ Type  │
├──────────────────────────────────────────┼─────────────────────────────┼───────┤
│ b.12345678-1234-1234-1234-123456789abc   │ My Company                  │ BIM360│
│ b.87654321-4321-4321-4321-cba987654321   │ Partner Organization        │ ACC   │
└──────────────────────────────────────────┴─────────────────────────────┴───────┘
```

## raps folder list

Показати вміст папки.

```bash
$ raps folder list b.proj-123 urn:adsk.wipprod:fs.folder:abc123
Folder Contents:
┌─────────┬────────────────────────────────┬───────────────────────────────┐
│ Type    │ Name                           │ ID                            │
├─────────┼────────────────────────────────┼───────────────────────────────┤
│ folder  │ Drawings                       │ urn:adsk...folder:def456      │
│ folder  │ Models                         │ urn:adsk...folder:ghi789      │
│ item    │ Project Schedule.xlsx          │ urn:adsk...item:jkl012        │
└─────────┴────────────────────────────────┴───────────────────────────────┘
```

**Показати папки верхнього рівня:**

```bash
raps folder list b.proj-123 --top
```

## raps folder create

Створення нової папки.

```bash
$ raps folder create b.proj-123 urn:adsk...folder:parent --name "New Folder"
✓ Folder created successfully!
  Name: New Folder
  ID: urn:adsk.wipprod:fs.folder:xyz789
```

## raps folder rename

Перейменування папки.

```bash
raps folder rename <PROJECT_ID> <FOLDER_ID> --name <NEW_NAME>
```

**Приклад:**

```bash
$ raps folder rename b.proj-123 urn:adsk...folder:abc123 --name "Revised Drawings"
✓ Folder renamed successfully!
```

## raps folder delete

Видалення папки.

```bash
raps folder delete <PROJECT_ID> <FOLDER_ID>
```

> **Попередження:** Папка повинна бути порожньою. Спочатку перемістіть або видаліть вміст.

**Приклад:**

```bash
$ raps folder delete b.proj-123 urn:adsk...folder:abc123
✓ Folder deleted successfully!
```

## raps item delete

Видалення елемента з проєкту.

```bash
raps item delete <PROJECT_ID> <ITEM_ID>
```

**Приклад:**

```bash
$ raps item delete b.proj-123 urn:adsk.wipprod:dm.lineage:abc123
✓ Item deleted successfully
```

## raps item rename

Перейменування елемента (оновлення відображуваної назви).

```bash
raps item rename <PROJECT_ID> <ITEM_ID> --name <NEW_NAME>
```

**Опції:**
- `--name, -n` - Нова відображувана назва

**Приклад:**

```bash
$ raps item rename b.proj-123 urn:adsk.wipprod:dm.lineage:abc123 --name "Updated Model v2.rvt"
✓ Item renamed to: Updated Model v2.rvt
```

## Типові робочі процеси

### Перегляд структури проєкту

```bash
# 1. Показати список хабів
raps hub list

# 2. Показати проєкти в хабі
raps project list b.12345678-1234-1234-1234-123456789abc

# 3. Показати папки верхнього рівня
raps folder list b.proj-123 --top

# 4. Переглянути конкретну папку
raps folder list b.proj-123 urn:adsk.wipprod:fs.folder:abc123
```

### Експорт даних проєкту

```bash
# Експорт усіх хабів у JSON
raps hub list --output json > hubs.json

# Експорт проєктів
raps project list $HUB_ID --output json > projects.json
```
