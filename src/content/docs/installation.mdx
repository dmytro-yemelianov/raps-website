# Installation

RAPS CLI can be installed using several methods. Choose the one that best fits your needs.

## Package Managers

### Cargo (Rust)

If you have Rust installed, this is the easiest method:

```bash
cargo install raps
```

**Requirements:** Rust 1.70+ ([rustup.rs](https://rustup.rs/))

### Homebrew (macOS/Linux)

```bash
brew tap dmytro-yemelianov/raps
brew install raps
```

### Scoop (Windows)

```powershell
scoop bucket add raps https://github.com/dmytro-yemelianov/scoop-raps
scoop install raps
```

## Pre-built Binaries

Download from the [Releases page](https://github.com/dmytro-yemelianov/raps/releases):

| Platform | Architecture | File |
|----------|--------------|------|
| Windows | x64 | `raps-windows-x64.zip` |
| macOS | Intel | `raps-macos-x64.tar.gz` |
| macOS | Apple Silicon | `raps-macos-arm64.tar.gz` |
| Linux | x64 | `raps-linux-x64.tar.gz` |
| Linux | ARM64 | `raps-linux-arm64.tar.gz` |

### Windows Installation

```powershell
# Download and extract
Expand-Archive raps-windows-x64.zip -DestinationPath "$env:USERPROFILE\bin"

# Add to PATH
$env:PATH += ";$env:USERPROFILE\bin"

# Make permanent
[Environment]::SetEnvironmentVariable("Path", $env:Path + ";$env:USERPROFILE\bin", "User")
```

### macOS/Linux Installation

```bash
# Extract the archive
tar -xzf raps-*.tar.gz

# Move to a directory in your PATH
sudo mv raps /usr/local/bin/

# Make executable
chmod +x /usr/local/bin/raps

# Verify installation
raps --version
```

## Docker

```bash
docker pull ghcr.io/dmytro-yemelianov/raps:latest

# Run with environment variables
docker run --rm \
  -e APS_CLIENT_ID="your_id" \
  -e APS_CLIENT_SECRET="your_secret" \
  ghcr.io/dmytro-yemelianov/raps:latest auth test
```

## GitHub Actions

```yaml
- name: Install RAPS
  uses: dmytro-yemelianov/raps-action@v1
  
- name: Use RAPS
  run: raps auth test
  env:
    APS_CLIENT_ID: ${{ secrets.APS_CLIENT_ID }}
    APS_CLIENT_SECRET: ${{ secrets.APS_CLIENT_SECRET }}
```

## Build from Source

```bash
# Clone the repository
git clone https://github.com/dmytro-yemelianov/raps.git
cd raps

# Build in release mode
cargo build --release

# Binary is at target/release/raps
```

## Verify Installation

```bash
raps --version
```

You should see output like:

```
raps 3.0.0
```
