# Cookbook для виробництва

Галузеві рецепти для виробничих процесів з використанням Inventor, Fusion, STEP, IGES та інших CAD-форматів.

## Рецепти

Оберіть рецепт, щоб розпочати роботу з детальними CLI-командами, CI/CD-конвеєрами та прикладами інтеграції з MCP.

### Обробка CAD

- [Конвеєр трансляції CAD](/cookbook/cookbook-mfg-translation) — Обробка різних CAD-форматів для веб-перегляду
- [Пакетна обробка кількох форматів](/cookbook/cookbook-mfg-batch) — Пакетна обробка каталогів з CAD-файлами

### Управління даними

- [Витяг специфікації матеріалів](/cookbook/cookbook-mfg-bom) — Витяг даних BOM зі збірок
- [Відстеження ітерацій проєктування](/cookbook/cookbook-mfg-iterations) — Організація CAD-файлів за ревізіями

### Автоматизація

- [Генерація креслень через DA](/cookbook/cookbook-mfg-drawings) — Генерація 2D-креслень за допомогою Design Automation
- [Створення пакетів для постачальників](/cookbook/cookbook-mfg-supplier) — Створення пакетів для передачі постачальникам

### Візуалізація

- [Генерація каталогу мініатюр](/cookbook/cookbook-mfg-catalog) — Побудова візуальних каталогів деталей
